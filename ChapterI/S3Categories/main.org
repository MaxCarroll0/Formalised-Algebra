* Categories

Definition 3.1: a category has objects and, for each pair of objects
\(A, B\), a set \(\mathrm{Hom}(A,B)\) of morphisms, with identities
\(1_A \in \mathrm{Hom}(A,A)\) and a composition
\(\mathrm{Hom}(A,B) \times \mathrm{Hom}(B,C) \to \mathrm{Hom}(A,C)\),
written \((f,g) \mapsto gf\), satisfying \(f 1_A = f\), \(1_B f = f\), and
\((hg)f = h(gf)\); distinct pairs of objects have disjoint morphism sets.
In the typed formalisations below disjointness is automatic, since
\(\mathrm{Hom}(A,B)\) are distinct types. The commuting square:

#+begin_export latex
\[\begin{tikzcd}
  A \arrow[r, "f"] \arrow[d, "g"'] & B \arrow[d, "h"] \\
  C \arrow[r, "k"']                & D
\end{tikzcd}\]
#+end_export

* Lean

The definition as stated in the book, with ~comp f g~ denoting \(gf\);
~one~ is the category with a single object and only its identity.

#+begin_src lean4 :tangle Lean/Content.lean
universe u v

structure Category where
  Obj : Type u
  Hom : Obj → Obj → Type v
  id : (A : Obj) → Hom A A
  comp : {A B C : Obj} → Hom A B → Hom B C → Hom A C
  id_right : ∀ {A B : Obj} (f : Hom A B), comp (id A) f = f
  id_left : ∀ {A B : Obj} (f : Hom A B), comp f (id B) = f
  assoc : ∀ {A B C D : Obj} (f : Hom A B) (g : Hom B C) (h : Hom C D),
    comp f (comp g h) = comp (comp f g) h

def one : Category where
  Obj := PUnit
  Hom _ _ := PUnit
  id _ := ⟨⟩
  comp _ _ := ⟨⟩
  id_right _ := rfl
  id_left _ := rfl
  assoc _ _ _ := rfl
#+end_src

* F*

The same definition with the laws as squashed propositions, followed by
Example 3.5: a monoid, here \((\mathbb{N}, +, 0)\), is a one-object
category.

#+begin_src fstar :tangle FStar/Content.fst
module Content

noeq type category = {
  obj: Type;
  hom: obj -> obj -> Type;
  id: (a: obj) -> hom a a;
  comp: (#a: obj) -> (#b: obj) -> (#c: obj) -> hom a b -> hom b c -> hom a c;
  id_right: squash (forall (a b: obj) (f: hom a b). comp (id a) f == f);
  id_left: squash (forall (a b: obj) (f: hom a b). comp f (id b) == f);
  assoc: squash (forall (a b c d: obj) (f: hom a b) (g: hom b c) (h: hom c d).
    comp f (comp g h) == comp (comp f g) h);
}

let nat_add : category = {
  obj = unit;
  hom = (fun _ _ -> nat);
  id = (fun _ -> 0);
  comp = (fun #_ #_ #_ f g -> f + g);
  id_right = ();
  id_left = ();
  assoc = ();
}
#+end_src
